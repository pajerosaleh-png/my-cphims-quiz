# Quick Deployment Guide

## Deploy in 2 Minutes with Vercel

```bash
# 1. Create a GitHub repository
# Go to https://github.com/new and create a repo named "cphims-quiz"

# 2. Push your code
git remote add origin https://github.com/YOUR_USERNAME/cphims-quiz.git
git branch -M main
git push -u origin main

# 3. Deploy with Vercel
# Go to https://vercel.com/new
# Click "Import Git Repository"
# Paste: https://github.com/YOUR_USERNAME/cphims-quiz
# Click "Import" then "Deploy"
# Done! Your site is live in 30 seconds
```

## Deploy with Netlify

```bash
# Same first 2 steps as above, then:
# Go to https://netlify.com/signup
# Click "New site from Git"
# Select GitHub and authorize
# Choose your cphims-quiz repository
# Click "Deploy site"
# Done!
```

## Deploy with GitHub Pages

```bash
# Same first 2 steps as above, then:
# Go to your repo Settings → Pages
# Select "Deploy from a branch"
# Choose "main" branch and "/root" directory
# Click "Save"
# Your site will be at: https://YOUR_USERNAME.github.io/cphims-quiz
```
